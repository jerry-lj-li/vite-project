```js
1, useCallback，useMemo 第二个参数是空数组[], 其执行机制是？
   -- 相当于每次都创建一个新的函数
2, 在函数组件中你要思考的方式永远是：当某个状态发生变化时，我要做什么，而不再是在 Class 组件中的某个生命周期方法中我要做什么。
```
